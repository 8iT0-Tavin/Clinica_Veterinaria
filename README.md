# 🐾 Clínica Veterinária São Camillo

Sistema desenvolvido como projeto de extensão da UEMG com o objetivo de organizar o gerenciamento de clientes e rotinas administrativas de uma clínica veterinária.
Conforme o Dono da clinica pediu por simplicidade na utilização 

---

## 📌 Sobre o Projeto

O sistema foi criado para resolver problemas reais de organização enfrentados pela clínica, substituindo controles manuais por uma solução digital simples, funcional e intuitiva.

Principais objetivos:
- Centralizar informações de clientes
- Melhorar a organização interna
- Facilitar o acesso a dados administrativos
- Tornar o atendimento mais eficiente

---

## 🚀 Funcionalidades

- Cadastro de clientes
- Edição e exclusão de registros
- Visualização organizada dos dados
- Interface simples e de fácil uso
- Organização de informações administrativas

---

## 🛠️ Tecnologias Utilizadas

- C#
- .NET
- Interface Desktop (Windows Forms)

---

## 🧠 Conceitos Aplicados

- CRUD (Create, Read, Update, Delete)
- Organização de dados
- Estruturação de interface para usabilidade
- Separação de responsabilidades
